# 🍰 BlissOven — Artisan Patisserie

An ultra-premium, interactive, and beautifully animated website for **BlissOven by Komal Agarwal**, a luxury handcrafted patisserie brand.

This project delivers a state-of-the-art sensory experience for choosing premium desserts, showcasing signature bakes, and streamlining order customization directly to WhatsApp.

---

## ✨ Features

- **🎭 Cinematic Hero Carousel:** High-impact slides showcasing seasonal specials, signature brownies, and custom celebration collections. Includes floating, organic product particles and smooth text animations.
- **📦 Interactive Brownie Box Builder:** A customized grid interface allowing customers to choose between a Mini Box (4 pcs), Gift Box (8 pcs), or Party Box (12 pcs), mix-and-match from 9 signature flavors, track selection progress, and compile a WhatsApp checkout template automatically.
- **🎨 Interactive Flavour Moodboard:** A sensory moodboard where hovering over each flavor (Chocolate Truffle, Strawberry, Rasmalai, etc.) dynamically shifts the section background, color scheme, and floating ambient particles.
- **👑 Royal Indian Fusion Collection:** A regal showcase of signature fusion bakes combining Indian mithai with European pastry techniques (Rasmalai Cake, Gulab Jamun Cake, Kesar Pista, and seasonal Alphonso Mango).
- **💌 Sweethearts Testimonial Carousel:** An automated, high-contrast feedback carousel featuring genuine ratings, location highlights, and clean layouts.
- **✨ Fluid Micro-Animations:** Floating ambient flour particles, parallax chef portraits, responsive hover filters, custom gold cursors, and magnetic button feedback.
- **📱 Responsive & Accessible Design:** Fully optimized across all viewport sizes with strict typography hierarchy, standard line spacing, and centered alignments for an organic layout.

---

## 🛠️ Tech Stack

- **Core Framework:** [Next.js (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Typography:** Playfair Display, Cormorant Garamond, Lora, and Inter via Google Fonts.

## 🚀 Getting Started

### Prerequisites

Ensure you have **Node.js** (v18.x or later) installed.

### Installation

1. Clone this repository:
   ```bash
   git clone <your-repository-url>
   cd blissoven
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the local development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

---

## 🏗️ Production Build

To compile a highly optimized static production bundle:

```bash
npm run build
```

Verify that the static pages render correctly and compile without hydration mismatch errors.
